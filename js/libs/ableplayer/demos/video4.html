Able Player Video Demo #4:
Video player with audio description via video swap

This demo uses a separate video deliver audio description. If Description is toggled on, the user gets the described version; otherwise they get the non-described version.

This method for delivering audio description, in contrast to the method used in Video Demo #3, has some limitations:

  • It cannot automatically pause when description is playing.
  • The description text is not included in the machine-generated interactive transcript.
  • Both the described and non-described versions of the video need to be the same duration.
