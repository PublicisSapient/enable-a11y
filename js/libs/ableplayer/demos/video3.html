Able Player Video Demo #3:
Video player with audio description via VTT track

This demo uses a WebVTT file to deliver text-based audio description. If Description is toggled on, supporting browsers will read the description text aloud using the Web Speech API. Video will pause temporarily while description is playing if the user checks "Automatically pause video when description starts" in Audio Description Preferences. The text from the description file is incorporated into the machine-generated interactive transcript.

For browsers that do not support the Web Speech API, description text will be written to an ARIA live region on the web page, and will be read aloud by screen readers.

For an alternative method of delivering audio description, see Video Demo #4.
