Able Player for YouTube (Simple Example)

This page features a relatively simple example of Able Player being used to play a YouTube video. Its features include:

  • A described and non-described version of the video, both hosted on YouTube. Swap between the two versions using the Description button.
  • Captions, subtitles, and a poster image, all hosted on YouTube.
  • An interactive transcript built from the caption and subtitle files.

For additional demos see the Index of Able Player Examples. In particular, check out the Complex YouTube Example.
