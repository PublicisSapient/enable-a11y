YouTube player (captions on YouTube, descriptions hosted locally)

This page features an example of Able Player being used to play a YouTube video. Its features include:

  • Captions and subtitles are hosted on YouTube. This video has been translated into a large number of languages, which makes it perfect for testing presentation and usability of such a long captions menu.
  • Descriptions and chapters are hosted locally as WebVTT files. This is necessary for text-based audio description since YouTube does not yet support this feature.
  • Description is turned on by default using data-state-descriptions="on".
  • Extended description (i.e., automatically pausing while description plays) is turned off by default using data-desc-pause-default="off". This is because the description for this video was carefully scripted to avoid conflicts with speakers in the program audio.
