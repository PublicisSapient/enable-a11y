ARIA Tooltip Example

JavaScript tooltips

Notes:

  • Tooltips should be visible onfocus, and be hidden onblur.
  • The widget that activates the tooltip should have an aria-describedby attribute that points to the tooltip so the AT can inform the user of it.

CSS only

Notes:

  • These tooltips are implemented using the HTML title attribute and CSS generated content (i.e. content: attr('title')).
  • Because of this, some screen-readers will read the tooltip twice.
  • Note on hover, the default user-agent tooltip will appear.
